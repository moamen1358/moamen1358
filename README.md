## About

I'm an AI Engineer at White Guard in Cairo. By day I build production
agentic systems — multi-agent runtimes, MCP servers, intelligence
pipelines. By night I publish small open-source tools that grew out
of problems I had at work.

&nbsp;

## Projects

&nbsp;

**[voci](https://github.com/moamen1358/voci)** — Real-time speech-to-text and translation overlay for Linux. Local NVIDIA Parakeet with optional cloud STT and LLM translation backends.

&nbsp;

**[f9-talk](https://github.com/moamen1358/f9-talk)** — System-wide hold-to-talk dictation for Linux. Single Rust binary distributed as a `.deb`.

&nbsp;

**[WannaScrape](https://github.com/moamen1358/WannaScrape)** — Production-grade web scraper with Playwright fingerprint rotation and plugin-based bot-detection.

&nbsp;

**[WannaSearch](https://github.com/moamen1358/WannaSearch)** — Google News search CLI and HTTP API with company-focused queries and per-company log archives.

&nbsp;

**[WannaClick](https://github.com/moamen1358/WannaClick)** — Browser extension that auto-clicks any text-matched element on any page.

&nbsp;

**[LUMINAFLOW](https://github.com/moamen1358/LUMINAFLOW)** — Desktop automation tool that records mouse, keyboard, and gesture input on Linux and replays it on demand.

&nbsp;

**[attendance-cv](https://github.com/moamen1358/attendance-cv)** — Face-recognition attendance system. YOLOv11 + ArcFace + ChromaDB on a Streamlit frontend.
