## Moamen Ghareeb

AI Engineer at White Guard, Cairo. I build voice-first developer tools, web automation pipelines, and content production systems.

### Active projects

- **[voci](https://github.com/moamen1358/voci)** — Real-time speech-to-text and translation overlay for Linux. Streaming Parakeet (NVIDIA NeMo) on the local GPU with optional cloud STT (AssemblyAI, Deepgram, Soniox) and cloud LLM translation (Cerebras, Gemini). Hybrid mode keeps live partials on local OPUS-MT while reserving the cloud LLM for committed sentences.
- **[f9-talk](https://github.com/moamen1358/f9-talk)** — System-wide hold-to-talk dictation for Linux. Single statically-linked Rust binary distributed as a `.deb`, with cloud or offline STT backends and an audio-reactive overlay.
- **[attendance-cv](https://github.com/moamen1358/attendance-cv)** — Computer-vision attendance system, originally my graduation project.
